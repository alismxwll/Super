This README will document cloning of this repository.

Getting started with cloning:

* open your terminal

* In your terminal, type "git clone https://github.com/andrewrdakers/Super.git"

* Then type, "bundle" to install all the required gems.

* Then type, "rake db:create" to create the database locally.

* Then type, "rake db:migrate" to populate the database with the correct tables and columns.

* Then type, "rails s" to start your server.
